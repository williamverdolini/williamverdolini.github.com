Abstract

Discitur is a web application for teachers that aimed to be a platform for sharing good and bad teaching practices, for analyzing pros and cons, in order to improve them, thanks to the usage by a lot of teachers in real life.
Besides that, it's a learning gym for Single Page Application development with Angular.js and for Agile software development tecniques.
